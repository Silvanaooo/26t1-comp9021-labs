## Best way to use this repo

1. **Attempt the lab questions first** (even partial attempts are valuable).
2. Compare with the solutions:
   - Identify what you did differently
   - Note the key idea / pattern used
3. Re-do the question **without looking** after 1‚Äì2 days.
4. For weak areas, use `Practice Questions/` and the LeetCode list.

If you only read solutions, improvement will be slow.
If you *try ‚Üí compare ‚Üí redo*, improvement will be fast.
